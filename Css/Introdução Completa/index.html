O que é Css

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo utilizada para descrever a apresentação de documentos escritos em HTML ou XML. Ela define a aparência visual de uma página web, controlando elementos como cores,fontes, espaçamento, layout e desing.

Sobre Nós Página Inicial

Existem 3 maneiras de aplicar CSS

inline

Na mesma linha, através de um atributo chamado style

Interno

Dentro do mesmo documento HTML, através de uma tag chamada Style

Externo

Criando um novo arquivo .css e utilizando seletores
